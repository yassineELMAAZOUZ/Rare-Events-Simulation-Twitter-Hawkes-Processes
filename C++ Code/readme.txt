#############
## Execution
#############
       

	-Compile the file main.cpp
	-Run the executable.
	-Choose x to fix the magnitude of probability to estimate:  (10^-x is will be the magnitude of the estimated probability) 
	-Choose number of estimations for the boxplot
	-Choose the simulation effort (size of Monte Carlo Sum)


         ------> a file estimations.txt will be generated in the folder, it contains the probability estimations.

	-Execute the python script plot.py to plot the box-plot and get estimation results.
